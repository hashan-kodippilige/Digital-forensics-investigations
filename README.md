# Digital Forensics Investigations

## Overview

This repository showcases hands-on digital forensics investigations completed as part of graduate-level Digital Forensics coursework.

The projects focus on evidence acquisition, forensic imaging, chain of custody, network forensics, packet analysis, incident response, timeline reconstruction, and cybercrime investigations.

The work follows digital forensic methodologies including evidence identification, preservation, collection, examination, analysis, and reporting.
